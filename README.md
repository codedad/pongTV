# pongTV
Introductory tvos implementation of the popular Pong game
* Xcode 7 / Swift 2.0
* tvOS 9.1
 
Features
=======
 - Classic pong gameplay
 - 1 player vs computer
 - 2 players (supported controllers: AppleTV Remote, Steelseries Nimbus)
